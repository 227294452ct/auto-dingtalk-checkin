# 自动钉钉打卡

Android 无障碍服务自动打卡应用，定时自动完成钉钉上下班打卡操作。

## 功能

- **定时打卡** — 设置上下班打卡时间，后台自动执行，每天准点打卡
- **节假日跳过** — 内置 2026 年中国法定节假日，周末/节假日自动跳过
- **立即打卡** — 支持手动触发，随时选择上班或下班打卡
- **锁屏唤醒** — 定时触发时自动唤醒屏幕并解除滑动锁屏，确保打卡流程不被锁屏阻断
- **失败通知** — 打卡失败时通过系统通知、短信或邮件提醒
- **打卡记录** — 历史打卡成功/失败记录查询
- **运行日志** — 内置日志查看器，支持按级别过滤，方便排查问题
- **开机自启** — 手机重启后自动恢复打卡服务

## 技术架构

- 语言：Kotlin
- 最低 SDK：Android 8.0 (API 26)
- 目标 SDK：Android 14 (API 34)

**核心模块：**

| 模块 | 说明 |
|------|------|
| DingDingAccessibilityService | 无障碍服务，识别钉钉页面元素并模拟点击完成打卡 |
| AutoCheckInService | 前台服务，通过 AlarmManager 精确定时唤醒 |
| AlarmReceiver | 闹钟广播接收器，触发打卡流程 + 锁屏唤醒 |
| HolidayManager | 节假日判断、周末/法定假日跳过 |
| CheckInRecordManager | 打卡记录本地 JSON 存储 |
| NotificationHelper | 失败多渠道通知（系统通知 / 短信 / 邮件） |
| LogManager | 内存环形日志缓存 + Logcat 双写 |

## 使用说明

1. 安装应用后，点击「开启辅助功能」，在系统设置中找到并启用「自动钉钉打卡」无障碍服务
2. 设置上班和下班打卡时间（默认 09:00 / 18:00）
3. 点击「开始自动打卡」，应用将在后台运行
4. 建议将应用加入电池优化白名单，避免被杀后台
