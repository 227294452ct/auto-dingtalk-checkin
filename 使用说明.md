# 自动钉钉打卡应用 - 使用说明

## 功能简介
本应用可以帮助您在 Android 手机上实现定时自动打卡，支持设置上班和下班打卡时间，到点自动打开钉钉并完成打卡操作。

## 安装步骤
1. 使用 Android Studio 打开项目
2. 等待 Gradle 同步完成
3. 连接 Android 手机或使用模拟器
4. 点击 "Run" 按钮安装应用

## 使用方法

### 1. 开启辅助功能
- 打开应用后，点击 "开启辅助功能" 按钮
- 在系统设置中找到 "自动钉钉打卡" 服务
- 开启该服务（需要授权访问屏幕内容的权限）

### 2. 设置打卡时间
- 设置上班打卡时间（默认为 9:00）
- 设置下班打卡时间（默认为 18:00）

### 3. 启动自动打卡
- 点击 "开始自动打卡" 按钮
- 应用将在后台运行，到设定时间自动执行打卡

## 技术原理

### 辅助功能服务
- 使用 Android AccessibilityService 监听并操作钉钉应用
- 自动识别页面元素并模拟点击操作
- 支持识别"工作台"、"打卡"、"确认"等按钮

### 定时任务
- 使用 AlarmManager 实现定时唤醒
- 每天在设定时间触发打卡操作
- 支持开机自启动

## 注意事项

### 权限要求
- **辅助功能权限**：必须开启，否则无法操作钉钉
- **后台运行权限**：请确保应用在后台不会被系统杀死
- **开机自启动**：建议开启，确保重启后仍能自动运行

### 手机设置建议
1. 将应用加入电池优化白名单
2. 关闭应用的后台限制
3. 确保钉钉已登录且处于正常状态

### 兼容性
- 最低支持 Android 8.0 (API 26)
- 建议使用最新版本的钉钉应用
- 由于钉钉界面可能会更新，应用可能需要适配新版本
